# easy-agent-mem v0.4.1

## Highlights

- Added tailored instruction files for Cursor, Claude, Antigravity, and OpenCode
- Strengthened the canonical memory workflow with explicit checks, summarization triggers, and anti-hallucination rules
- Updated the instruction generator so `agent-mem init` produces the same detailed templates automatically

## What Changed

### IDE-specific instructions

- Cursor now gets a dedicated `.mdc` rule focused on memory-first execution, milestone summaries, and conflict checks
- Claude / VS Code gets stronger session-start and summarization instructions
- Antigravity and OpenCode now receive more detailed wrapper instructions instead of placeholder files

### Canonical rules

- `AGENT-MEM-RULES.md` was expanded into a real source of truth
- Added explicit rules for:
  - memory priority
  - session start behavior
  - summarization timing
  - pre-save double checks
  - Obsidian-specific behavior
  - fallback-mode behavior

## Verification

- generated instruction files were recreated and checked for expected content
- packaging and build artifacts passed verification

pip install -U easy-agent-mem
