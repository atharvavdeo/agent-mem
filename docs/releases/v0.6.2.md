# easy-agent-mem v0.6.2

## Highlights

- Polished graph dashboard output with stronger navigation and clearer operational signal
- Added source-breakdown and actionable suggestion sections to Graph-Report
- Improved concept confidence clarity for deterministic vs inferred entries
- Fixed duplicated unreachable graph build code path

## What Changed

### Graph dashboard polish

- Enhanced `Index.md` with generation timestamp, table of contents, and quick navigation links
- Added direct project-root README links from the graph dashboard output
- Improved CLI graph output summary to print the dashboard entry file path

### Graph report quality

- Added source breakdown metrics (memory source channels, chat snippets, tagged decision/blocker comments)
- Added suggestion bullets based on blockers, decision coverage, enrichment mode, and compact mode

### Concept readability

- Added confidence guide clarifying:
  - `EXTRACTED` concepts are deterministic and display `n/a`
  - `INFERRED` concepts include LLM confidence when enrichment is enabled

### Reliability

- Removed duplicated unreachable graph generation block in `build_graph`
